# ユーザー認証

アプリのユーザー認証機能

## サブ機能一覧

| サブ機能 | 説明 |
|----------|------|
| アカウント登録 | アカウントを作成 |
| ログイン | ユーザ認証 |

## 画面一覧

| 画面名 | URL | ルート名 |
|--------|-----|----------|
| アカウント登録 | `/user-auth/register` | `user-auth.register` |
| ログイン | `/user-auth/login` | `user-auth.login` |

## 画面遷移図

| 操作 | 画面遷移 |
|------|--------|
| ※ルート | [root] --> ホーム |
| アカウント登録 | ホーム --> アカウント登録 ==> ホーム |
| ログイン | ホーム --> ログイン ==> ホーム |
