# アカウント管理

webサービス利用アカウントを管理

## サブ機能一覧

| サブ機能 | 説明 |
|----------|------|
| アカウント登録 | アカウントを作成 |
| ログイン | メールアドレスとパスワードで認証 |
| ログアウト | セッションを破棄してログアウト |
| パスワードリセット申請 | メールでリセットリンクを送信 |
| パスワード再設定 | メールのリセットリンクから接続し再設定 |

## 画面一覧

| 画面名 | URL | ルート名 |
|--------|-----|----------|
| アカウント登録 | `/auth/signup` | `auth.signup` |
| ログイン | `/auth/login` | `auth.login` |
| パスワードリセット申請 | `/account/forgot-password` | `account.forgot-password` |
| パスワードリセット再設定 | `/account/reset-password/{token}` | `account.reset-password` |

## 画面遷移図

| 操作 | 画面遷移 |
|------|--------|
| ※ルート | [root] --> ホーム |
| ログイン | ホーム --> ログイン ==> ホーム |
| ログアウト | ホーム --> [logout] ==> ホーム |
| アカウント登録 | ログイン --> アカウント登録 ==> ホーム |
| パスワードリセット | ログイン --> パスワードリセット申請 --> [e-mail] --> パスワードリセット再設定 ==> ログイン |
